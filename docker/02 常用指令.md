# 🛠️ Docker 常用指令

## Container 管理

```bash
# 查看容器
docker ps              # 執行中的容器
docker ps -a           # 所有容器（含停止的）

# 啟動容器
docker run nginx                        # 基本啟動
docker run -d nginx                     # 背景執行
docker run -d -p 8080:80 nginx          # 對外開 port（本機:容器）
docker run -d --name my-app nginx       # 幫容器命名
docker run -it ubuntu bash              # 互動模式進入容器

# 停止 / 刪除
docker stop <container>        # 優雅停止
docker kill <container>        # 強制停止
docker rm <container>          # 刪除容器
docker rm -f <container>       # 強制刪除（執行中也可）
docker container prune         # 刪除所有停止的容器
```

---

## Image 管理

```bash
# 查看 image
docker images
docker image ls

# 下載 image
docker pull nginx
docker pull nginx:1.25          # 指定版本

# 刪除 image
docker rmi nginx
docker image prune              # 刪除沒用到的 image
docker image prune -a           # 刪除所有沒有容器在用的 image

# 建立 image
docker build -t my-app:1.0 .        # 從當前目錄的 Dockerfile 建立
docker build -t my-app -f Dockerfile.prod .   # 指定 Dockerfile
```

---

## 進入容器 / 查 Log

```bash
# 進入執行中的容器
docker exec -it <container> bash
docker exec -it <container> sh      # 有些容器沒有 bash，用 sh

# 查看 log
docker logs <container>
docker logs -f <container>          # 持續追蹤 log（像 tail -f）
docker logs --tail 50 <container>   # 只看最後 50 行
```

---

## 查看資訊

```bash
docker inspect <container>          # 容器詳細資訊（JSON）
docker stats                        # 即時資源使用（CPU、Memory）
docker top <container>              # 容器內的行程
docker port <container>             # 查看 port 對應
```

---

## 系統清理

```bash
docker system df                    # 查看 Docker 佔用空間
docker system prune                 # 清理所有未使用資源
docker system prune -a              # 更激進的清理（含 image）
```

---

## 💡 常用組合技

```bash
# 停止並刪除所有容器
docker stop $(docker ps -q) && docker rm $(docker ps -aq)

# 刪除所有 image
docker rmi $(docker images -q)

# 進入容器後執行指令
docker exec -it my-app bash -c "ls /app"
```
